The Missing Marks Project is a web-based system designed to help students, teachers, and administrators track, manage, and report missing or unrecorded exam marks. The system ensures accurate recording and easy retrieval of student performance data.

Features
User Roles: Supports multiple roles such as Student, Teacher, and Admin.
Login & Signup: Secure authentication system for all users.
Marks Reporting: Teachers can submit missing marks for exams or assignments.
Student Dashboard: Students can view their reported marks and track status.
Admin Panel: Admins can manage users, verify reported marks, and generate reports.
Notifications: Alerts users when missing marks are added or updated.# Miss-Marks-Projects
